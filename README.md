# 🛡️# 🛡️ Smart TV + Roku Telemetry & Ad Blocklist (Pi-hole / DNS)

A curated collection of DNS blocklists designed to reduce ads, telemetry, tracking, and background data collection on:

- Roku devices
- Samsung Smart TVs
- LG webOS TVs
- Vizio Smart TVs
- Android/Google TV
- Streaming apps (YouTube, Prime Video, etc.)

> ⚠️ Note: Some platforms (especially YouTube) may still serve ads due to first-party delivery systems. DNS blocking cannot fully remove all video ads.

---

## 🚀 Features

- Blocks Smart TV telemetry endpoints
- Reduces tracking from streaming devices
- Blocks major ad networks
-

How to use 
1-Log in to your Pi-hole Admin Console.

2-Go to Lists 

3-Paste the URL of the blocklist (https://github.com/thatmikeman1/Pihole-roku-youtube-DNS-blocklist.txt) into the Address box.

Click Add

Refresh Pi-hole's active list (Gravity) to enforce the changes by going to Tools > Update Gravity and clicking Update
